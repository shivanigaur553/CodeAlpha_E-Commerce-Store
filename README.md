# 🛒 E-Commerce Store

A responsive and interactive E-Commerce Store built using HTML, CSS, and JavaScript. This project simulates a real online shopping platform where users can register, log in, browse products, manage their cart, create a wishlist, and view product details.

## 🚀 Features

* User Registration & Login Authentication
* Secure Access to Store Pages
* Product Search Functionality
* Category-Based Product Filtering
* Product Details Page
* Shopping Cart Management
* Quantity Increase/Decrease Controls
* Wishlist Functionality
* User Profile Page
* Bill Summary with GST Calculation
* Checkout System
* Responsive and User-Friendly Interface
* Local Storage for Data Persistence

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📖 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/shivanigaur553/CodeAlpha_E-Commerce-Store.git
   ```

2. Open the project folder in VS Code.

3. Run the project using Live Server or open `home.html` in your browser.

## 🎯 Key Functionalities

* Register a new account and log in.
* Browse products with images, ratings, and prices.
* Search products by name.
* Filter products by category.
* Add products to cart and wishlist.
* View detailed product information.
* Generate bill summary with GST and delivery charges.
* Manage user profile information.
* Checkout and place orders.
